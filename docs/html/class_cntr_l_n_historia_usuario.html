Descrição detalhada

Controladora de logica de negocio das historias de usuario.

Mantem a associacao de cada historia com um projeto ou com um plano de sprint (relacionamento exclusivo) e garante que a soma das estimativas das historias de um plano nao exceda a capacidade do plano.
